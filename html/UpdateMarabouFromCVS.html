How to download or update MARaBOU from CVS

  • login as user mbuser, password marabou
    cvs -d :pserver:mbuser@cvs.mll-muenchen.de:/home/cvs/marabou login
  • if it is first time you access MARaBOU's cvs server:
    cvs -d :pserver:mbuser@cvs.mll-muenchen.de:/home/cvs/marabou checkout marabou
  • to look for recent changes
    cvs -d :pserver:mbuser@cvs.mll-muenchen.de:/home/cvs/marabou -n update 2>/dev/null | grep '^[MPUC]'
  • to update an existing MARaBOU installation:
    cvs -d :pserver:mbuser@cvs.mll-muenchen.de:/home/cvs/marabou update

How to compile MARaBOU after download or update

  • Adjust file iniMrbPath in MARaBOU's top directory (SYSMODE=y, MARABOU=`pwd`, ROOTSYS set properly)
  • Execute ". iniMrbPath" to activate environment settings
  • Do a make to recompile MARaBOU
  • Maybe a make install is needed to make it accessible to others
  • Provide a modified iniMrbPath for users (SYSMODE=n, MARABOU=MARaBOU's top dir)
