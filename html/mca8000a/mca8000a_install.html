Installation and running

For reasons of flexibility the software is currently implemented using ROOTs ACLiC (Automatic Compiler for Libraries) facility.

This software and the needed changes in the TMrbSerialCom class are implemented only from Root/Marabou versions 5.34.10 onwards. So at the moment (23.10.2013) you should do

module unload marabou module load marabou/5.34.10

To use it you should copy the following files into an empty directory:

$MARABOU/macros/mca8000/mca8000.h $MARABOU/macros/mca8000/mca8000.cxx $MARABOU/macros/mca8000/rootlogon.C

Then start HistPresent. At this moment rootlogon.C will be executed which builds (first time) and loads the shared library, creates an instance of the MCA8000 class and starts a GUI. For more info press the Help button.

If you want to run HistPresent without executing rootlogon.C use option -n
HistPresent -n
