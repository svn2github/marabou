Programs/Tools to convert MLLE data

  • mhdr
    • Outputs MLLE header data to stdout
    • May be used for raw data files as well as histograms
    • Syntax:
      • mhdr filename
      • mhdr /dev/dat
  • mtcopy
    • Copies MLLE raw data
      • from tape to disk
      • from tape to (remote) tape
    • Syntax: see manual page
  • mlle2hbook
    • Converts MLLE data to CERN's HBOOK format
      • MLLE raw data to NTuples
      • MLLE histograms to HBOOK histograms
    • Syntax: see manual page
  • rdtio.o
    • Functions to process MLLE raw data
    • Provides a means to
      • open/close MLLE raw data files
      • decode records
      • decode events
      • show headers
    • For details see /home/rg/dcnv/lsrc/rdtio.c
  • ospio.o
    • Functions to read MLLE histogram data
    • Provides function calls to
      • open/close an OSP histogram file
      • read histgram data, 1-dim or 2-dim
      • show header info
    • For details see /home/rg/dcnv/lsrc/ospio.c
