ASCII to histograms

The following input formats are handled to convert data given as numbers (double) in a plain ASCII file to a histogram:
  • Channel contents

    In this case the N values in the ASCII file are assumed to be the contents of the 1 - N channels of the spectrum.
    A histogram with N bins, low egde 0, upper edge (double)N is created.

  • Channel contents, Errors

    Same as above, the second value is the bin error.

  • X values of histogram

    A histogram is created and incremented at X with weight 1

  • X values, weights

    Same as above, the histogram incremented at X with weight .

  • X-, Y-values of 2dim histogram

    A 2-dim histogram is created and incremented at X, Y with weight 1

  • X-, Y-values, weights of 2dim histogram

    A 2-dim histogram is created and incremented at X, Y with weight.

In all cases (hopefully) reasonable defaults are provided for the histogram parameters.

ASCII to graphs

The following input formats are handled to convert data given as numbers (double) in a plain ASCII file to a TGraph or TGraphErrors:
  • X, Y of TGraph (without errors)

    A TGraph with X, Y values is created.

  • X, Y, ErrorX, ErrorY of TGraphErrors

    A TGraphError with X, Y, ErrorX, ErrorY values is created.

  • X, Y, ErrorXlow, ErrorXhigh, ErrorYlow , ErrorYhigh of TGraphAsymmErrors

A TGraphAsymmErrors with X, Y, ErrorX, ErrorY values is created. Graphs may be drawn in the same window. To use this feature use the file popup menu of the canvas into which the graph should go and select the option: Draw in same window
