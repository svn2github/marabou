Usage of the mouse (pointer)

In general mouse button 2 (middle) is used to set marks used in fitting and window setting operations. Button 3 (right) triggers a popup menu provided by the selected object. Actions triggered by mouse button 1 (left) depend on the object below the pointer:
  • If it points to a command button obviously the command is executed if it points to a name in a Filelist a list of contents will be presented etc.
  • If the pointer is inside the frame of a histogram pressing and dragging button 1 will popup a window showing bin-numbers, x -y values and content of the current bin and the integral in the dragged region. Optionally a fit of Gaussian plus a linear background is done in real time when moving the mouse. This option can be switched on and off from the option item: How to display a 1-dim histogram or How to display a 2-dim histogram This feature may also temporarily switch off and on for the current canvas from the ROOT-Options popup item Pad Auto Exec
  • If the pointer is inside the X- Y- or Z-scale of a histogram pressing button 1 and dragging will zoom into the dragged region. Unzoom may be done invoking the popup menu by button 3.
  • If the pointer is inside the statistics box its content is printed to the transcript window.
  • If the pointer is inside a canvas with several histograms the selected histogram is shown in a separate window. This is described in more detail here
