Overview

HistPresent was developed within the Marabou Project to give simple access to ROOT objects typically used in Accelerator experiments.
To use HistPresent at Garching on the MLL-Linux-Cluster type the following commands:

set_rootnew to set PATHs etc.
Execute HistPresent in a directory where your ROOT-files reside.

Outside Garching please consult the installation Howto

Browsing objects in files, memory, memory mapped files

  • Histograms (1, 2 or 3-dim)
  • Ntuples and simple Trees
  • Graphs
  • Contours (levels in 2-dim histograms)
  • Cuts, Windows (1-dim, 2-dim)
  • Functions
  • Canvases

Display and manipulate histograms

  • expand, zoom
  • project, profile
  • superimpose
  • add extra axis
  • highlight regions
  • add, subtract, multiply, divide
  • lin / log scale
  • display many histograms in one canvas
  • save (manipulated) histograms to ROOT files
  • output to printers, ps, gif files

Graphical cuts

  • define cuts in 1 and 2-dim hists
  • apply, activate - deactivate
  • save / restore cuts to / from ROOT files

Fitting histograms

  • Gauss + tails + background
  • Polynomials
  • user defined macros using templates
  • save / restore functions to / from ROOT files

Calibrating 1-dim histograms

  • linear by simple change of scale
  • non linear: rebin, derandomize

Conversion between flat (ASCII) files and ROOT objects

  • ASCII data to 1, 2, 3-dim histograms
  • 1 - dim histograms to ASCII
  • ASCII data to Graphs (with or w/o errors)

Access to many Style options

  • Drawing options
  • Canvas sizes, arrangments
  • Linestyles, Fillstyles, Colors, GFonts
  • Axis attributes, Margins

An interface to ROOTs graphics editor + extras

  • Provide standard edit canvases (Portrait, Landscape, User)
  • Implement an user defined edit grid
  • Insert histograms, graphs, functions
  • Insert images (TASImage)
  • Insert LaTex from keyboard or file
  • Implement Xsplines
  • Global editing features
  • Implement compound objects
  • Save / restore picture
